Bem-vindo ao Website Pitágoras Calc!

Este site foi cuidadosamente desenvolvido como parte de uma atividade avaliativa sob a orientação do professor Helder Cavalcante, como parte do curso de Análise e Desenvolvimento de Sistemas na Universidade Pitágoras Unopar Anhanguera.

Nosso objetivo é fornecer a você uma ferramenta eficaz para calcular seu salário líquido e rescisão de férias de forma simples e precisa. Com a ajuda das linguagens de programação que aprendemos neste curso, criamos uma plataforma que facilitará seus cálculos financeiros.

Navegue pelo site para acessar as diferentes calculadoras disponíveis e descubra como podemos simplificar seus processos de cálculo. Este projeto representa nosso comprometimento com a excelência acadêmica e a aplicação prática dos conhecimentos adquiridos.

Fique à vontade para explorar todas as funcionalidades do site. Agradecemos por visitar nosso projeto e esperamos que ele seja útil para você!
